## Abbreviations

| Abbreviation | Full Name | Description |
|---|---|---|
| AC | Acceptance Criteria | Criteria for accepting a deliverable |
| GIP | Global Inviolable Principle | Principles that must never be violated |
| DoD | Definition of Done | Criteria that define when work is complete |

---

## Term Usage Rules

1. **Consistency**: Use the same term for the same concept across code, documentation, and comments
2. **Unambiguous**: Avoid using words with multiple meanings
3. **Traceable**: New terms must be defined in this document before use
