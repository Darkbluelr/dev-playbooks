# Bootstrap Spec Template

This template is used to document baseline artifacts and output checklist during brownfield project bootstrap.

## Minimal Artifact Set

- Project profile
- Glossary
- Key concepts
- Baseline specs
- Minimal validation anchors

## DoR（Definition of Ready）

| Condition | Decision | Handling |
| --- | --- | --- |
| Baseline artifacts complete | Must satisfy | If missing, trigger completion |
| Change scope is clear | Must satisfy | If unclear, request clarification |
| Risk labeling is available | Must satisfy | If missing, record as a blocker |

## Validation Anchors

- Output path: `dev-playbooks/specs/_meta/`
- Evidence location: `dev-playbooks/changes/<change-id>/evidence/`
