# /devbooks:impact

Triggers Skill: `devbooks-impact-analysis`

## Purpose

Perform impact analysis before cross-module/cross-file/external contract changes, producing the Impact section that can be directly written to proposal.md.

## Typical Use Cases

- Assessing impact scope before changes
- Identifying affected modules and APIs
- Analyzing compatibility risks

Please invoke `devbooks-impact-analysis` to execute the corresponding task.
