# /devbooks:gardener

Triggers Skill: `devbooks-spec-gardener`

## Purpose

Pre-archive pruning and maintenance of truth-root (deduplication/merging, removing outdated content, directory organization, consistency fixes) to prevent specs from accumulating out of control.

## Typical Use Cases

- Cleanup before archiving a change package
- Specs directory needs deduplication and organization
- Cleaning up outdated specification documents

Please invoke `devbooks-spec-gardener` to execute the corresponding task.
